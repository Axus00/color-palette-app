# 🎨 Color Palette App  
🎨 Una aplicación para generar, visualizar y guardar paletas de colores.  

## 🚀 Tecnologías Usadas  
- 🟣 Backend: .NET 8, Entity Framework Core  
- 🔵 Frontend: React, TailwindCSS  
- 🛠️ Base de Datos: MySQL

## 📦 Instalación  

### 🔹 Backend (.NET)
```bash
cd backend
dotnet restore
dotnet run
```

### 🔹 React
```bash
cd frontend
npm install
npm start
```
